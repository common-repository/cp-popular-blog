== Description ==
This is a customized plugin to display the top 5 popular posts of any custom post type you create. Configure your widget and it will start storing the post as popular in the backgound.


== Installation ==
1. Upload the extracted zip to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
